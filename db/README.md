# Neon PostgreSQL Database Setup

## Running the SQL Files

1. Open your Neon project dashboard at [console.neon.tech](https://console.neon.tech) and navigate to the **SQL Editor**
2. Create a new query and copy the entire contents of `schema.sql`, then click **Run**
3. Create another new query and copy the entire contents of `seed.sql`, then click **Run**
4. Your database is now initialized with the required tables and initial bankroll record

The dashboard and bot can now read/write data as configured.
